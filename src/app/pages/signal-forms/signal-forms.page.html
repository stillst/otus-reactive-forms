Signal Forms

Stable API since Angular 22. Uses form(), [formField] directive, and schema-based validation from @angular/forms/signals.

Basic Signal Form

form(signal, schemaFn) creates a typed field tree. Bind fields with [formField]. Access state via field().value(), field().errors(), etc.

Custom Validation (Cross-field)

Use validate(path, fn) with valueOf(otherPath) to compare fields reactively. A custom error is a plain { kind, message } object.

Conditional Logic (disabled / hidden)

disabled(path, { when }) disables a field reactively. hidden(path, { when }) hides it — hidden fields skip validation. Since v22 the condition goes into the { when: ... } config; passing a function directly is deprecated.
